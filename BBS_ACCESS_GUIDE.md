# 📌 掲示板（BBS）アクセスガイド

## 🎯 掲示板機能の利用方法

### ✅ アクセス方法

掲示板機能は**ログイン不要**で、誰でも自由に利用できます！

#### 方法1: ヘッダーメニューから
1. アプリケーションを起動（`run.bat`を実行）
2. ブラウザで http://localhost:8080 にアクセス
3. **ヘッダーメニューの「掲示板」をクリック**

```
┌─────────────────────────────────────┐
│ ⚽ 海外サッカーニュース              │
│                [ホーム] [掲示板] [最新] │ ← ここをクリック！
└─────────────────────────────────────┘
```

#### 方法2: 直接URLにアクセス
ブラウザのアドレスバーに以下を入力：
```
http://localhost:8080/bbs
```

## 📝 掲示板でできること

### 1️⃣ スレッド一覧を見る
- カテゴリ別にフィルタリング
- キーワード検索
- 人気スレッドをチェック

### 2️⃣ 新規スレッドを作成する（匿名投稿）
1. 「✏️ 新規スレッド作成」ボタンをクリック
2. 以下の情報を入力：
   - **カテゴリ**: プレミアリーグ、ラ・リーガなど
   - **タイトル**: スレッドのタイトル
   - **本文**: 投稿内容
   - **名前**: ニックネーム（匿名可）
   - **パスワード**: 削除用（4文字以上）
3. 「📝 スレッドを作成」をクリック

**ログイン不要！** 名前とパスワードを入力するだけで投稿できます。

### 3️⃣ スレッドを読む・コメントする
1. スレッドタイトルをクリック
2. 本文とコメントを読む
3. 下部のフォームからコメント投稿
   - **名前**: ニックネーム
   - **コメント**: 返信内容
   - **パスワード**: 削除用（4文字以上）
4. 「💬 投稿する」をクリック

### 4️⃣ おすすめ（イイネ）する
- スレッド詳細画面で「👍 おすすめ」ボタンをクリック

### 5️⃣ 投稿を削除する
- 「🗑️ 削除」ボタンをクリック
- 投稿時に設定したパスワードを入力

## 🔐 セキュリティ機能

### ✅ 実装済みのセキュリティ対策
- **XSS対策**: 悪意のあるスクリプトは自動でエスケープ
- **CSRF対策**: クロスサイトリクエストフォージェリを防止
- **パスワード保護**: BCryptで暗号化（削除時に必要）
- **SQLインジェクション対策**: 完全に防止

### ⚠️ 注意事項
- 削除用パスワードは忘れないようにメモしてください
- パスワードは4文字以上必要です
- 誹謗中傷や個人情報の投稿は禁止です

## 💡 よくある質問（FAQ）

### Q1: ログインは必要ですか？
**A:** いいえ、ログイン不要です。名前とパスワードを入力するだけで投稿できます。

### Q2: 匿名で投稿できますか？
**A:** はい、名前欄に「匿名」など好きなニックネームを入力できます。

### Q3: 投稿を削除するには？
**A:** 投稿時に設定したパスワードが必要です。「🗑️ 削除」ボタンからパスワードを入力してください。

### Q4: パスワードを忘れた場合は？
**A:** 残念ながら、パスワードを忘れると削除できません。投稿時にメモしておくことをお勧めします。

### Q5: スレッドが見つからない場合は？
**A:** キーワード検索やカテゴリフィルタを使って探してみてください。
